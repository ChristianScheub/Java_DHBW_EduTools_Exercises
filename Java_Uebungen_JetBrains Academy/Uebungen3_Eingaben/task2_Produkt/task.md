## Produkt der ersten n Zahlen

Schreiben Sie ein Java-Programm, das ein Integer n als Befehlszeilenargument entgegennimmt und das Produkt n! = 1*2*3*…*n bildet.

Konkret soll in der Main Methode mit dem Scanner die Zahl entgegen genommen werden mit der Funktion scanner.nextInt(). Diese soll dann in eine andere Funktion übergeben werden.
Erstelle Sie also die Funktion calculateFactorial welche die Fakultät der übergebenen Zahl berechnet und übergibt.
Anschließend soll das Ergebnis von der Main Methode ausgegeben werden.
