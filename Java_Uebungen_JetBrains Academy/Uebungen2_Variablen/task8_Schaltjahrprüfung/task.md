## Schaltjahrpruefung

Implementieren Sie die Java-Methode `findLeapYear(int year)`, die eine Jahreszahl als Variable übergeben bekommt, ob dieses Jahr nach dem Gregorianischen Kalender einem Schaltjahr entspricht.

Tipp: Mit dem Befehl `return` können sie Variablen aus einer Funktion zurückgeben. In dem Fall soll ein `boolean` zurückgegeben werden, ob es sich um ein Schaltjahr handelt.
