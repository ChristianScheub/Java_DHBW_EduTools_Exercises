# HalloWelt-Programm

In dieser Übung werden Sie die Grundlagen der Java-Entwicklung üben, indem Sie ein einfaches Java-Programm erstellen, kompilieren und ausführen.

**Schritte für die Kommandozeile:**

1. Öffnen Sie einen Texteditor Ihrer Wahl.
2. Geben Sie das folgende Java-Programm ein:

    ```java
    public class HalloWelt {
        public static void main(String[] args) {
            System.out.println("Hallo Welt!");
        }
    }
    ```

3. Speichern Sie die Datei unter dem Namen `HalloWelt.java`.
4. Öffnen Sie die Kommandozeile (Command Prompt oder Terminal) und navigieren Sie zu dem Verzeichnis, in dem Sie die Datei gespeichert haben.
5. Kompilieren Sie das Programm mit dem Befehl:

    ```
    javac HalloWelt.java
    ```

6. Führen Sie das kompilierte Programm mit dem Befehl aus:

    ```
    java HalloWelt
    ```

Beobachten Sie, was im Terminal ausgegeben wird.

**Schritte für die Entwicklungsumgebung (z. B. Eclipse oder NetBeans):**

1. Starten Sie Ihre Entwicklungsumgebung.
2. Erstellen Sie ein neues Java-Projekt und eine neue Klasse mit dem Namen `HalloWelt`.
3. Geben Sie den gleichen Java-Code wie oben in die Klasse ein.
4. Verwenden Sie die integrierten Funktionen der Entwicklungsumgebung, um das Programm zu kompilieren und auszuführen.

Beachten Sie die Unterschiede zwischen der Verwendung der Kommandozeile und der Entwicklungsumgebung in Bezug auf Benutzerfreundlichkeit, Geschwindigkeit und Komfort.

Nachdem Sie das Programm sowohl über die Kommandozeile als auch über die Entwicklungsumgebung ausgeführt haben, beantworten Sie die Quizfrage, um Ihr Verständnis zu überprüfen.
